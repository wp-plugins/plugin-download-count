== Installation ==

1. Upload the `plugin-download-count` folder to the `/wp-content/plugins/` directory
2. Active the Download Count plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Download Count` menu that appears in your admin menu
4. Insert the [PDC] shortcode on any Page, Post or Custom Post Type to display the total download count
